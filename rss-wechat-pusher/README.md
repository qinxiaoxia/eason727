# RSS → 企业微信推送（需求版）

将 RSS 订阅（WeWe RSS + 网安站点）自动推送到企业微信群，支持分类、实时/定时推送。

> 新手请先看 **[入门指南.md](./入门指南.md)**。

## 功能

| 功能 | 说明 |
|------|------|
| **多源** | 微信公众号（WeWe RSS）+ 7 个网安 RSS |
| **分类** | 6 类：监管机构预警、漏洞信息、重大安全事件、网安新闻资讯、网安赛事资讯、其他资讯 |
| **规则分类** | 监管机构预警：公众号+标题规则（Phase 1 已实现） |
| **大模型分类** | 其余类别（Phase 2 待接入） |
| **轮巡实时** | 监管机构预警、重大安全事件 → 仅在北京 6:00–20:00 每两小时整点运行中推送 |
| **定时推送** | 漏洞信息等其余四类 → 仅在北京 9:30、15:30 汇总推送 |
| **推送格式** | 【信息类型】【标题】【时间】【链接】 |
| **去重** | SQLite 记录已推送，避免重复 |

## 文件说明

| 文件 | 说明 |
|------|------|
| `main.py` | 主程序 |
| `classifier.py` | 分类器（规则 + 大模型占位） |
| `config.py` | 配置（需自行创建） |
| `config.example.py` | 配置示例 |
| `rss_push.db` | 文章与推送记录（自动生成） |

## 运行

```bash
pip3 install -r requirements.txt
python3 main.py
```
